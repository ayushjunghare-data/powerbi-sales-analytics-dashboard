# 📊 Power BI Sales & Customer Analytics Dashboard

---

## 🚀 Project Overview
This project presents an **end-to-end Business Intelligence dashboard** developed using **Power BI** to analyze **sales performance, customer behavior, and product trends**.

The objective was to transform raw business data into **interactive visual insights** that support strategic decision-making.

---

## 🧩 Business Problem
The organization lacked a centralized reporting system to monitor:

- Revenue performance  
- Profit trends  
- Order behavior  
- Customer insights  

Decision-makers were relying on fragmented reports, making analysis slow and inefficient.

---

## 💡 Solution
Designed an **interactive and scalable Power BI dashboard** using:

✅ Star Schema Data Model  
✅ Advanced DAX Measures  
✅ Dynamic Visual Analytics  

The dashboard enables stakeholders to track KPIs and uncover business insights in real time.

---

## 🏗️ Data Modeling
- ⭐ Star Schema Architecture  
- 📦 1 Fact Table — **Sales**
- 🧾 Dimension Tables:
  - Customer
  - Product
  - Date
  - Territory
- 🧠 Separate **Measure Table** created for centralized business logic

---

## ✨ Key Features
- 📈 Executive KPI Dashboard  
- 🎯 Target vs Actual Performance Analysis  
- 👥 Customer Segmentation Analysis  
- 🛍️ Product Performance Insights  
- 🌍 Regional Sales Analysis  
- ⚡ Dynamic Metric Selection using **Field Parameters**

---

## 🛠️ Tools & Technologies
- 📊 Power BI  
- 🧮 DAX (Data Analysis Expressions)  
- 🧱 Data Modeling  
- 🔄 Power Query (ETL & Data Transformation)

---

## 🔍 Key Business Insights
- 📈 Revenue increased despite a slight drop in orders — indicating **higher Average Order Value**
- 🥇 Accessories category generated the highest sales volume
- ⚠️ Tires & Tubes category showed relatively higher return rates
- 💰 High-value customers contributed a major share of total revenue
