# Power BI Sales & Customer Analytics Dashboard

## Project Overview
This project presents an end-to-end Business Intelligence dashboard developed in Power BI to analyze sales performance, customer behavior, and product trends.

## Business Problem
The business lacked a centralized system to track revenue, profit, orders, and customer insights for decision-making.

## Solution
Designed an interactive dashboard using Power BI with a Star Schema data model and advanced DAX measures.

## Data Modeling
- Star Schema Architecture
- 1 Fact Table (Sales)
- Multiple Dimension Tables (Customer, Product, Date, Territory)
- Separate Measure Table for business logic

## Key Features
- Executive KPI Dashboard
- Target vs Actual Analysis
- Customer Segmentation
- Product Performance Analysis
- Regional Sales Insights
- Dynamic Metric Selection using Field Parameters

## Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Power Query

## Key Insights
- Revenue increased despite slight drop in orders indicating higher average order value
- Accessories category drove highest sales volume
- Tires & Tubes showed higher return rate
- High-value customers contributed significant revenue

Data Modeling | DAX | Business Analytics | Dashboard Design | Data Storytelling
